# Solemate Web Project

## Project info

This is the source code for Solemate website.

## How can I edit this code?

There are several ways of editing your application.

**Use your preferred IDE**

1. Open the project folder in your favorite IDE (e.g., Visual Studio Code, Sublime Text, Atom).
2. Edit the files as you see fit. The main files for the front end are located in the `src` directory.
3. Save your changes and see them reflected in the browser if you have the development server running.

**Use the command line**

If you prefer working with the command line, you can also edit the files using command-line text editors like `vim` or `nano`.

1. Navigate to the project directory using `cd path/to/your/project`.
2. Use a command-line text editor to open and edit the files. For example, `vim src/App.vue`.
3. Save your changes and exit the editor. If you are using `vim`, you can do this by pressing `Esc`, then typing `:wq` and hitting `Enter`.

## How can I run this code?

To run the project, you need to have [Node.js](https://nodejs.org/) installed on your machine.

1. Clone the repository to your local machine using `git clone https://github.com/yourusername/your-repo.git`.
2. Navigate to the project directory using `cd path/to/your/project`.
3. Install the dependencies by running `npm install`.
4. Start the development server with `npm run dev`.
5. Open your browser and go to `http://localhost:3000` to see the project in action.
